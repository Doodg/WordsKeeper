# WordsKeeper
Android App that takes note and encrypts it for a specific period then unencrypted it and display it in the notification.

![](https://i.imgur.com/DqKb6Fs.gif)

## Work Flow
- Homepage (list of notes)
- Add new note
- Write note title
- Select unlock time
- Click Save

## Libraries Used
- AndroidX
- ViewModel
- Room
- Paging
- Worker
- Stetho
